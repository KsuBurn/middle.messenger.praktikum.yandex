## Установка

- `npm install` — установка зависимостей
- `npm run dev` — запуск версии для разработки
- `npm run start` — билд проекта и запуск express
- `npm run build` — билд проекта
- `npm run code-check` - запуск проверки eslint, stylelint и TS
